# Portfolio Web - Victor Vallejo

Portfolio personal desarrollado para presentar mi perfil como Fullstack Developer y Software Engineer, con foco en rendimiento, mantenibilidad y valor de negocio.

La web incluye version en español e inglés, una sección de presentación personal y una sección de experiencia profesional con contenido gestionado de forma desacoplada.

## Que puedes ver en este proyecto

- Home con propuesta de valor y enfoque tecnico
- Seccion About Me con contexto personal y profesional
- Seccion Experience con trayectoria y stack por etapa
- Cambio de idioma ES/EN
- Interfaz moderna, limpia y orientada a lectura rápida

## Tecnologias

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- react-icons
- Zod
- sanitize-html

## Objetivo

Este repositorio no busca ser solo una demo visual: esta pensado como una base profesional para evolucionar un portfolio escalable, con contenido validado y una estructura preparada para crecimiento.
