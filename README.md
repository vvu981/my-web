# Portfolio Web - Victor Vallejo

Portfolio personal desarrollado para presentar mi perfil como Fullstack Developer y Software Engineer, con foco en rendimiento, mantenibilidad y valor de negocio.

La web incluye version en español e inglés, una sección de presentación personal y una sección de experiencia profesional con contenido gestionado de forma desacoplada.

- Interfaz moderna, limpia y orientada a lectura rápida
